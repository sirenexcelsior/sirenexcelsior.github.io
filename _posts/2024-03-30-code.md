## Task of Assembly Language - Lecture 1

### 0 (Example)

```assembly
MOV AX,0	; Move 0 to register AX, i.e., assign the value
END
```

### 2000

Write a program that puts 2000 in the AX register.

```assembly
MOV AX, 2000
END
```

### Data

Write a program that puts data from BX in the AX register.

```assembly
MOV AX, BX
END
```
